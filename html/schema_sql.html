schema_sql {logR}R Documentation

logR schema scripts dictionary

Description

Use logR_schema to execute all scripts at once.

Usage

schema_sql(schema = getOption("logR.schema"),
  table = getOption("logR.table"), meta = list())

Arguments

schema

character schema name for log storing.

table

character table name for log storing.

meta

list of metadata columns, postgres data types as character string named by column name like list(tag='VARCHAR(255)', val_col='DOUBLE PRECISION', int_col='INTEGER', systimecol='TIMESTAMPTZ'). Avoid reserved postgres keywords.

See Also

logR_schema, logR

Examples

sql = schema_sql(schema="logr")
invisible(lapply(sql, sapply, cat, sep="\n"))
