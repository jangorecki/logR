update_make_set {logR}R Documentation

Make SET for update statement

Description

Helper for producing update statement, wraps character types into single quotes, numbers as non scientific.

Usage

update_make_set(col, x)

Arguments

col

character vector of column names to update.

x

data.table a one row data.table.

Value

character vector which can be easily paste(x,collapse=",") to paste in update statement.
